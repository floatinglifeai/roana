关于

Roana 漫行,把一部智能手机——以及未来的智能眼镜——变成一个静静陪伴你出门的伙伴。端侧计算机视觉看着前方的路。世界以两种方式回到你身边:骨传导语音(告诉你周围有什么)和腕部震动(告诉你该往哪走)。耳朵留给世界本身。

名字来自 roam——想去哪儿就去哪儿的自由和尊严。

我们【是】和【不是】什么

是

  • 电子辅助出行工具,增强白盲杖和导盲犬
  • 多模态:语音和触觉,协同工作
  • 基于今天就能买到的货架硬件
  • 开源,AGPL-3.0 协议

不是

  • 医疗器械
  • 白盲杖、导盲犬、定向行走训练的替代品
  • 视力恢复产品
  • 有付费墙的商业服务

它用起来什么感觉

语音
骨传导耳机里有一个安静的声音,只告诉你重要的事情:"前方有台阶,两米"、"右边是斑马线"。耳道是开放的,世界的声音听得见。
震动
左手或右手腕短促一震,引导你走向安全的路径。两只手同时震,就是停下。学会大概只需要一个小时。
按需
按一下按钮:"前面是什么?" 一个更大的模型仔细看一眼,然后告诉你。其他时候,它静静地在那儿,看着。

为什么开源

盲人和低视力群体被这种事情伤害过太多次:一个本来开放或免费的辅助技术,被收购、闭源、加上付费墙,再卖回给他们。Roana 选择 AGPL-3.0 协议,这是我们能给的最强保证——任何修改后的版本,包括上线的服务,都必须以同样的协议公开。

代码在 GitHub 上,任何人可以自由学习、自由 fork、自由用它来做事。
