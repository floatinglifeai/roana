# Domain and Trademark Landscape (Naming Round 2)

> The "roam / free exploration" naming direction landscape and how we ended up with **Roana**. Records why Roam / Rove / Vela / Pace / Voya etc. weren't usable, and what the alternatives are if Roana ever has to be retired.

> **Note (May 2026):** `roana.app` was acquired by the team after this research. Domain and trademark clearance for Nice classes 9 / 10 / 42 still needs formal counsel review before commercial launch — see checklist at the end of this doc.

---

## TL;DR — Top Candidates Considered

Final shortlist after Round 2 (focused on "roam / free exploration" semantic field + domain available + clean trademark):

1. **Roana / 漫行 (màn xíng)** — **chosen.** 3-syllable smooth, screen-reader clear; almost no same-name tech / accessibility / navigation brands on web; multiple suffixes likely registrable. Chinese 漫行 directly inherits "roam," concise, screen-reader friendly, no pity tone.
2. **Wendr / 漫途 (màn tú)** — high recommendation. The 2010-founded NYC nightlife app of the same name closed on Feb 8, 2012 (farewell post explicit "On Wednesday, February 8th, 2012"), namespace effectively freed for 14 years.
3. **Andora / 行远 (xíng yuǎn)** — high recommendation with reservation. From Italian *andare* (to go/walk); no tech-circle collision. **One firm cost: permanent spelling-confusion with the country "Andorra."**

**Key reminder:** all "registrable" conclusions in this doc are **inferred** from public web visibility, not registrar real-time responses. Before signing off any name, three things must be done: ① registrar real-time WHOIS query; ② trademark search in CN/US/EU (USPTO/EUIPO/CNIPA, at least Nice classes 9 / 10 / 42); ③ Google Play & App Store search check.

## Three: Other Empowering-Tone Directions — Mostly Wiped Out

To prevent re-exploration of the same dead ends, briefly:

- **"Freedom / Liberation":** Libre / Libero — taken by LibreOffice, **Abbott FreeStyle Libre** (continuous-glucose-monitoring medical brand, FreeStyle Libre 2/3, US CGM mainstream). Not available.
- **"Road / Departure":** Via / Avia squeezed by navigation/aviation; Sojo (sojo.app.uk is a UK clothing alteration platform) taken.
- **"Step / Walk":** Stride / Stryd (Stryd is famous running power meter, stryd.com), Stryde (StrydeBike), etc. nearly all taken.
- **"Explore":** Aira is aira.io — **same-domain visually-impaired remote-vision service**, per product page *"Aira can be used free of charge at more than 50 airports, universities, and major retailers like Walmart, Starbucks, and Target"*; Walmart 2024-12-03: *"Walmart is now offering the wayfinding service entirely free to customers"* — **direct same-domain collision, absolutely avoid.**
- **"Compass":** Compass / Kompas heavily taken.
- **Conclusion:** **"Roam-root + soft suffix"** (the Roana / Andora / Andala line) is the only realistic path that simultaneously satisfies tone + domain + trademark.

### 🥉 #3 Backup — Andora / 行远

- **English:** Andora (/ænˈdɔːrə/, Italian *andare* "to go/walk" root)

- Pro: very few collisions / Con: spelling slightly more complex than Roana

## Seven: Caveats (must be stated honestly)

1. **All "registrable," "free," "no collision found" conclusions in this doc are inferred from public web search as of May 2026,** not from registrar WHOIS real-time. WHOIS is free, 5 min to confirm. **Please verify yourself.**
2. **Domain status changes in real time.** A domain that looks free today may be sniped tomorrow (especially .app / .ai — squatters monitor new trademark filings and GitHub repo names). **After confirming preference, register immediately — even just 1 year.**
3. **Trademark search has professional technical threshold.** Nice classification, similar-mark determination, industry-category coverage all need professional judgment. If you go commercial (not just open-source), strongly recommend engaging a professional trademark agency for freedom-to-operate search (typically ¥3000–8000, worth it).
4. **Class 10 especially important:** visually-impaired assistive hardware (bone-conduction headsets, vibration wristbands, smart glasses temples) falls in Nice class 10 (medical devices and assistive equipment). HumanWare, Aira, Be My Eyes, Envision all registered here — scrutinize carefully.
5. **Open-source + commercialization dual track:** project name ownership needs to be clarified up-front in GitHub Org charter / LICENSE / CONTRIBUTING.md (reference Linux Foundation, CNCF, Mozilla project governance templates) to avoid future commercialization blocks or contributor brand-equity claims.
6. **Naming sensitivity in the visually-impaired community:** after the name is decided, do at least 5 visually-impaired users' voice-playback test — have iOS VoiceOver and Android TalkBack read "Roana / 漫行" in both Mandarin and English. Verify: ① audibly clear; ② voice-search returns correct result; ③ no negative associations; ④ easy to spread by word-of-mouth with cane / dog peers. Low-cost, high-value step.
7. **Research limited by search budget** — not all B/C-tier candidate suffixes verified. Andala / Roanna / Andova / Vialux / Viato domain statuses especially need user self-check.
8. **If all top 5 candidates turn out taken at real-time query**, can do another round of focused "roam-root + soft-suffix" pure-coined output (e.g., Roavi, Andaro, Wenova, Roanai, Mantia, Roalia) — but note coined words = weaker semantic association, needing stronger product narrative and visual system to support.

## Recommendations (Decision Trigger Conditions)

**Acted upon:** Roana / 漫行 chosen; roana.app registered; GitHub org floatinglifeai/roana created.

**Triggers for falling back to alternative names:**
- If **formal trademark clearance returns blocked in any of class 9, 10, or 42** in any of the three target jurisdictions → switch to Andala (purely coined, safest from trademark perspective).
- If **>2 of 5 visually-impaired users in voice-playback test report "can't hear clearly / mishear"** → switch to Wendr (shorter syllables) or re-name.
- If a **major collision emerges** (someone files identical mark with priority, or new product launches in assistive-tech with confusingly similar name) → activate defensive measures + assess re-name necessity.

**6-month retrospective (after GitHub repo live):** observe whether community feedback shows misspelling, search-not-found, hard-to-remember signals — these are real signals of naming quality. Concurrently watch Google Trends for any new same-name tech company; if Roana takes off, defensively register .org / .net / .me.
